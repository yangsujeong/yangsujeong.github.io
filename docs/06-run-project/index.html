6. 프로젝트 구동

  • 터미널에서 다음 커맨드를 실행하여 node 서버를 구동 한다.
# application 구동
yarn run dev:bo

구동중 표시되는 url을 통해 정상 구동여부를 확인한다.

6. 1. 프로젝트 구동 확인​

프로젝트 구동 작업에서 확인한 url로 접속 시 로그인 화면이 표출되며 admin / admin 으로 로그인 한다.

정상적으로 로그인후 Home 화면이 표시되는지 확인한다.
