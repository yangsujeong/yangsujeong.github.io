# **6. 프로젝트 구동**

- 터미널에서 다음 커맨드를 실행하여 node 서버를 구동 한다.

```bash
# application 구동
yarn run dev:bo
```

구동중 표시되는 url을 통해 정상 구동여부를 확인한다.

![image.png](/img/guide/image12.png)
